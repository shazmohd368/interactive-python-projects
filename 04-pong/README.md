# Pong – Classic Arcade Game

Mini project from:
"An Introduction to Interactive Programming in Python" – Rice University

## Description

Implementation of the classic Pong arcade game using the SimpleGUI framework.

Two players control paddles and attempt to bounce the ball past the opponent's paddle to score points.

## Controls

Player 1:
- W → Move paddle up
- S → Move paddle down

Player 2:
- ↑ → Move paddle up
- ↓ → Move paddle down

Restart button resets the game.

## Concepts Used

- Event-driven programming
- Collision detection
- Velocity and reflection physics
- Paddle boundary constraints
- Score tracking
- SimpleGUI graphics framework

## How to Run

This project uses the `simplegui` module in CodeSkulptor.

To run:
1. Open CodeSkulptor
2. Paste `main.py`
3. Click Run

⚠️ Note: `simplegui` is not part of standard Python and will not run locally without modification.
